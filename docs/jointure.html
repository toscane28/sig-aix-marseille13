Jointures spatiales

Introduction

Cette page présente la jointure spatiale entre les carreaux Filosofi (200m) et les IRIS, permettant d’agréger les revenus à une maille administrative.

Schéma de géotraitement

filosofi.gpkg ──┐
                ├──► st_join (st_within) ──► revenu par carreau + CODE_IRIS
iris.gpkg     ──┘                        ──► aggregate (médiane) par IRIS
                                         ──► merge avec géométries IRIS
                                         ──► iris_revenus  ──► carte choroplèthe

Jointure spatiale carreaux → IRIS

# Joindre chaque carreau à l'IRIS qui le contient
# On garde seulement les colonnes utiles de iris
join <- st_join(
  filosofi[, c("ind_snv", "geom")],
  iris[, c("code_iris", "nom_iris", "nom_commune", "geom")],
  join = st_within
)

cat("Carreaux joints à un IRIS :", sum(!is.na(join$code_iris)), "\n")
Carreaux joints à un IRIS : 2785 
cat("Carreaux sans IRIS        :", sum(is.na(join$code_iris)), "\n")
Carreaux sans IRIS        : 2896

Agrégation du revenu médian par IRIS

df <- st_drop_geometry(join)
df <- df[!is.na(df$code_iris) & !is.na(df$ind_snv), ]

revenu_iris <- aggregate(
  df$ind_snv,
  by  = list(code_iris = df$code_iris),
  FUN = median,
  na.rm = TRUE
)
names(revenu_iris)[2] <- "revenu_median"

# Fusionner avec les géométries IRIS
iris_rev <- merge(iris, revenu_iris, by = "code_iris")

cat("IRIS avec données de revenu :", nrow(iris_rev), "\n")

Conclusion

La jointure spatiale confirme le contraste entre les deux villes : Aix-en-Provence présente des IRIS avec des revenus médians globalement plus élevés, tandis que Marseille montre une forte hétérogénéité, avec des IRIS très pauvres concentrés dans les quartiers nord et des IRIS aisés dans les arrondissements sud.
